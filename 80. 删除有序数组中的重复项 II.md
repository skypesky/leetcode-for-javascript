
# [80. 删除有序数组中的重复项 II](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-array-ii/)

- 双指针

```typescript
function removeDuplicates(nums: number[]): number {

  const { length } = nums;

  let fast = 2, slow = 2;

  while (fast < length) {
    if (nums[fast] !== nums[slow - 1] || nums[fast] !== nums[slow - 2]) {
      nums[slow] = nums[fast];
      slow++;
    }
    ++fast;
  }

  return slow;

};
```
