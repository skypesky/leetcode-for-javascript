# [15. 位1的个数](https://leetcode.cn/problems/number-of-1-bits/)

- 位运算

```typescript
/**
 * @description 利用 n & (n - 1) 消除最低位的 1。
 *              每次消除一位 1，直到 n 变为 0，消除的次数即为 1 的个数。
 *              例如 n = 11 (二进制 1011)：
 *              - n = 1011, n-1 = 1010, n & (n-1) = 1010
 *              - n = 1010, n-1 = 1001, n & (n-1) = 1000
 *              - n = 1000, n-1 = 0111, n & (n-1) = 0000
 *              共消除 3 次，所以 11 有 3 个 1。
 *
 * 时间复杂度 O(k) 推导：
 * - k 为 n 二进制中 1 的个数
 * - 每次循环消除一个 1，共循环 k 次
 *
 * 空间复杂度 O(1) 推导：
 * - 仅使用常数级变量 count
 */
function hammingWeight(n: number): number {
    let count = 0;

    while (n) {
        n = n & (n - 1);
        ++count;
    }

    return count;
};
```
