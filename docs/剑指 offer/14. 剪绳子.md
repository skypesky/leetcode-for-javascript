# [14. 剪绳子](https://leetcode.cn/problems/integer-break/)

- 动态规划

```typescript
/**
 * @description 动态规划：dp[i] 表示将正整数 i 拆分成至少两个正整数后的最大乘积。
 *              状态转移：对于每个 i，在位置 j（1 <= j < i）处拆分
 *              - j * (i - j)：仅拆分为两部分，j 和 i-j 不继续拆分
 *              - f[j] * (i - j)：j 继续拆分，i-j 不拆分
 *              - j * f[i - j]：j 不拆分，i-j 继续拆分
 *              - f[j] * f[i - j]：两部分都继续拆分
 *              取四种情况的最大值即为 dp[i]
 *
 * 时间复杂度 O(n²) 推导：
 * - 外层循环 i 从 3 到 n，共 n-2 轮 → O(n)
 * - 内层循环 j 从 1 到 i-1，累计 1+2+...+(n-1) = n(n-1)/2 → O(n²)
 * - 总复杂度 O(n²)
 *
 * 空间复杂度 O(n) 推导：
 * - dp 数组 f 大小为 n+1
 * - 仅使用常数级额外变量 max, i, j
 * - 总空间 O(n)
 */
function integerBreak(n: number): number {
    const f: number[] = new Array(n + 1).fill(0);
    f[2] = 1;

    for (let i = 3; i <= n; ++i) {
        let max = 0;
        for (let j = 1; j < i; ++j) {
            max = Math.max(
                j * (i - j),
                f[j] * (i - j),
                j * f[i - j],
                f[j] * f[i - j],
                max
            );
        }
        f[i] = max;
    }

    return f[n];
};
```
