# [16. 数值的整数次方](https://leetcode.cn/problems/shu-zhi-de-zheng-shu-ci-fang-lcof/)

## 快速幂

```typescript
/**
 * @description 快速幂（递归实现）：
 *              - 当 n = 0 时，任何数的 0 次方为 1
 *              - 当 n > 0 时，将 n 折半计算 x^(n/2)
 *              - 如果 n 是奇数，结果需要再乘以一个 x
 *              - 如果 n 是偶数，结果就是 half^2
 *              - 递归公式：f(n) = f(n/2)^2 (n为偶数) 或 f(n) = f(n/2)^2 * x (n为奇数)
 *              - 负数幂转换为正数幂：x^(-n) = 1 / x^n
 *
 * 时间复杂度 O(log n) 推导：
 * - 每次递归将 n 折半，递归深度为 log n
 * - 每层递归只做常数次操作（一次乘法）
 * - 总操作次数 = log n
 *
 * 空间复杂度 O(log n) 推导：
 * - 递归栈深度为 log n（每次将 n 折半）
 * - 每层递归只需要常数级变量
 */
function myPow(x: number, n: number): number {
    function doPow(x: number, n: number): number {
        if (n === 0) return 1;
        const half = doPow(x, Math.floor(n / 2));
        return n % 2 === 0 ? half * half : half * half * x;
    }
    return n >= 0 ? doPow(x, n) : 1 / doPow(x, -n);
}
```
