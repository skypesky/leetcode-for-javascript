# [17. 打印从1到最大的 n 位数](https://leetcode.cn/problems/da-yin-cong-1dao-zui-da-de-nwei-shu-lcof/)

## 解法一：暴力法（简单解法）

```typescript
/**
 * @description 算法思路描述：
 * 1. 根据位数 cnt 计算出最大值为 10^cnt
 * 2. 创建一个数组，容量为 max - 1
 * 3. 使用循环从 1 遍历到 max-1，将每个数字存入数组
 * 4. 返回数组
 *
 * 时间复杂度 O(10^n) 推导：
 * - 需要遍历并存储从 1 到 10^n - 1 的所有数字
 * - 共计 10^n - 1 个数字，因此时间复杂度为 O(10^n)
 *
 * 空间复杂度 O(10^n) 推导：
 * - 需要创建一个长度为 10^n - 1 的数组来存储所有数字
 * - 因此空间复杂度为 O(10^n)
 *
 * 注意：当 n 较大时（如 n > 3），10^n 会超出 JavaScript Number 的安全整数范围，
 * 可能导致精度问题。这种情况下需要考虑大数表示。
 */
function countNumbers(cnt: number): number[] {
    const max: number = Math.pow(10, cnt);
    const res: number[] = new Array(max - 1);
    for(let i = 1; i < max; ++i) {
        res[i - 1] = i;
    }

    return res;
};
```
