
# [665. 非递减数列](https://leetcode-cn.com/problems/non-decreasing-array/)

- 放大还是缩小呢

```
function checkPossibility(nums: number[]): boolean {
    if (!nums || nums.length < 3) {
        return true;
    }

    let count: number = 0;
    for (let i = 1; i < nums.length; ++i) {
        if (nums[i - 1] <= nums[i])
            continue;
        ++count;
        if (count > 1) {
            return false;
        }
        if (i > 1 && nums[i - 2] > nums[i]) {
            nums[i] = nums[i - 1];
        } else {
            nums[i - 1] = nums[i];
        }
    }
    return true;
};
````
