
# [164. 最大间距](https://leetcode-cn.com/problems/maximum-gap/)

- 直接排序

```ts
function maximumGap(nums: number[]): number {

    if (nums.length < 2) {
        return 0;
    }

    nums.sort((a, b) => a - b);
    return nums.reduce((max: number, value: number, index: number): number => {
        index && (max = Math.max(nums[index] - nums[index - 1], max));
        return max;
    }, 0);
};
```
