
# [215. 数组中的第K个最大元素](https://leetcode-cn.com/problems/kth-largest-element-in-an-array/)

- 库函数

```ts
/**
 *
 * @complex T(nlogn)/S(1);
 * @param {number[]} nums
 * @param {number} k
 * @return {*}  {number}
 */
function findKthLargest(nums: number[], k: number): number {
    nums.sort((a, b) => b - a);
    return nums[k - 1];
};
```
