# [206. 反转链表](https://leetcode-cn.com/problems/reverse-linked-list/)

- 迭代法

```javascript
/**
 * @思路: 循环
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var reverseList = function (head) {
    let newHead = null,
        current = head;

    while (current) {
        const next = current.next;

        current.next = newHead
        newHead = current;
        
        current = next;
    }

    return newHead;
};
```
