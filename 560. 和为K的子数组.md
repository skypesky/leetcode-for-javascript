
# [560. 和为K的子数组](https://leetcode-cn.com/problems/subarray-sum-equals-k/)

- 暴力解法

```typescript
function subarraySum(nums: number[], k: number): number {

    let count: number = 0;

    for (let left = 0; left < nums.length; ++left) {
        let sum: number = 0;
        for (let right = left; right >= 0; --right) {
            sum += nums[right];
            if(sum === k) {
                ++count;
            }
        }
    }

    return count;
};
```
